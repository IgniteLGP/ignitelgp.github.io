About

Existem muitas plataformas de eCommerce online, quer via canal web tradicional (browser), quer via canal mobile (iPhone, Android, etc.). Estas plataformas são conceptualmente muito semelhantes, mas tendem a ser proprietárias dos respetivos lojistas, e resultam de investimentos consideráveis. E.g.: http://www.elcorteingles.pt/app_eci/index.asp, http://www.continente.pt/pt-pt/public/Pages/Apps/listas-continente.aspx.


Daqui surge uma potencial oportunidade: a criação de uma plataforma de eCommerce inteiramente mobile e adaptável a qualquer negócio. Ou seja, serão 2 Apps: um backoffice para o lojista gerir o seu catálogo, encomendas, etc.; e uma mobile store para clientes. Ambas comunicarão com uma plataforma de serviços comum. Este setup poderá ser adaptado graficamente e disponibilizado ao cliente sobre uma plataforma de serviços dedicada, ou seja, uma instalação para cada cliente, possível alojada em cloud (https://azure.microsoft.com/en-us/services/cloud-services/). As Apps deverão também tirar partido das últimas tecnologias de desenvolvimento híbridas multiplataforma (https://www.visualstudio.com/explore/cordova-vs), para aceleração do desenvolvimento.
